JS數據類型

發表於 2018-12-14

ECMAScript中有7種數據類型：number, string, boolean, undefined, null, symbol(ES6)

目錄：

  • Number 數值
  • String 字串
  • Boolean 布爾值
  • undefined 未定義
  • null 空值
  • Object 物件/(我會以對象來描述)

Number 數值

在JS內部，所有數字都是以64位浮點數形式存儲來表示所有數值。

JS底層根本沒有整數，所有數字都是浮點數

最基本數值字面格式：

數值類型是使用國際標準IEEE754格式來表示整數和小數點，為了支持各種數值類型，ECMA-262定義了不同數值字面格式。

JS會自動把二，八，十六進制轉換十進制

數值範圍

由於內存限制，ECMAScript不能保存世界上數值。
ECMAScript能夠表示最小數值保存在Number.MIN_VALUE在大多數browser中，這個值5e-324

ECMAScript能夠表示最小數值保存在Number.MAX_VALUE在大多數browser中，這個值1.7976931348623157e+308

如果在計算中結果超過JS數值範圍，那麼這個值將自動轉成特殊Infinity值。具體說明，如果這個值是負會轉成-Infinity(負無窮)，如果這個值是正會轉成Infinity(正無窮)。
