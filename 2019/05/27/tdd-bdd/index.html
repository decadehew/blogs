這篇讓我們來學習 TDD 和 BDD 是什麼。

首先先了解這兩個 term 概念

TDD (Test Driven Development)

TDD 是測試驅動開發，也就是先寫測試再寫程式，步驟：

  • 先寫一個測試 test case
  • 接下來寫功能程式
  • run test -> pass or failed
  • 重構 (Refactor), 添加新 test case
